#Getting and Building
1. Clone the repo
```bash
git clone https://github.com/swdhahn/bird-engine $SOURCE_DIR
cd $SOURCE_DIR
```
2. Run the cmake file. CMAKE_BUILD_TYPE can be Release or Debug.
```
cmake -DCMAKE_BUILD_TYPE=Debug -B $BUILD_DIR
cd $BUILD_DIR
cmake --build .
cmake .
```
